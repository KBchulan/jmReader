# JmReader

    如你所见，这个是基于jmcomic做的二次开发，主要原网站广告多，而且存在一些令人不喜的题材，作者这里做了个个人用的，为此学习了vue，果然，瑟瑟是第一生产力（大嘘），这里贴一个效果的网址：http://120.27.201.149:5173/

## 配置

    说不来被封了呢，有需要的可以自己跑一下玩玩

### 开发环境配置

前端：
```
npm install  
npm run dev
```

后端：
```
pip install -r requirements.txt  
./start.sh
```
